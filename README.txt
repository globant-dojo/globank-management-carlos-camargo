Script Sql

1. Abrir el script de SQL SERVER BaseDatos.sql con el software management studio teniendo una instancia de Sql Server disponible
2. Ejecutar el archivo BaseDatos.sql

Contenedor Docker

1. Descargar el codigo del repositorio correspondiente a ccamargoo (git clone https://github.com/globant-dojo/globank-management-carlos-camargo.git)
2. Cambiar la cadena de conexion del archivo appsettings.json
3. (En visual studio) Dar clic derecho sobre el proyecto BankDojo y dar clic en la opcion publicar
4. Seccionar el ambiente o folder de publicacion del contenedor
5. Configurar el contenedor en el servicio Docker del servidor
